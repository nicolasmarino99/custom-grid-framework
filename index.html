YAGF (Yet Another Grid Framework) demo

by Nico & Oscar, the "Dream Team" :-)

This framework has two different modes:

Bootstrap mode: In this mode, the layout will mimic the behaviour of Bootstrap, using selectors similar to that framework, but with the variant that you select the starting column and the ending column for the cell.

Grid mode: In this mode, the user can select the cells using an Excel-like notation. The default grid size is 12 columns by 3 rows, but that can be changed by modifying the respective values from the defs.scss fragment and recompiling the framework.
